In Chapter 7:

StoreModule is a NgModule which uses ProductRepository and StaticDataSource through dependency Injection. As we add StoreModule to [Imports] of app.module.ts, we don't need to specify dependecies such as ( StoreComponent, ProductRepository and StaticDataSource ). We can get the same implementation without creatintg a module with following code.



Initial code:
--------------------------------------------------------
--------------------------------------------------------
@NgModule({
  declarations: [
    AppComponent
  ],
  imports: [
    BrowserModule,StoreModule
  ],
  providers: [],
  bootstrap: [AppComponent]
})

--------------------------------------------------------
--------------------------------------------------------

New code:
--------------------------------------------------------
--------------------------------------------------------
@NgModule({
  declarations: [
    AppComponent, StoreComponent
  ],
  imports: [
    BrowserModule
  ],
  providers: [ProductRepository, StaticDataSource],
  bootstrap: [AppComponent]
})
